## 技术栈

## 功能介绍

```properties
台管理员功能有个人中心，用户管理，民宿信息管理，民宿分类管理，我的收藏管理，系统管理，订单管理等。

用户可以注册登录，查看民宿和预定名宿,以及加入购物车, 查看民宿资讯和在线咨询客服。
```
